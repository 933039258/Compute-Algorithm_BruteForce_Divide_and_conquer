==Coordinate Generator:=====
Type the command "gcc generator.cpp -lstdc++ -o generator -lm" on any linux machine, 

then run the command "./generator",

Then the generator will generate 40 file contain 10 of each level of input number, from 10^2 to 10^5.
============================


==Enhanced Divide Function:===
Type the command "gcc enhancedDivide.cpp -lstdc++ -o enhancedDivide -lm" on any linux machine, 

then run the command "./enhancedDivide input_file_name",

Then the program will launch and output a file with run time and coordinates, 

name of output file is "output_enhanceddnc.txt".
==============================
